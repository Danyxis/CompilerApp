# CompilerApp
# Руководство пользователя приложения «Компилятор»

## 1. Пункт меню «Файл»

- **Создать** — очистка рабочей области для создания нового документа. Предыдущие изменения сохраняются при необходимости.
- **Открыть** — загрузка текста из выбранного файла. Перед открытием программа предложит сохранить изменения, если они есть.
- **Сохранить** — сохранение текущего документа. Если файл ещё не был сохранён, будет предложено выбрать путь.
- **Сохранить как** — сохранение документа с выбором имени файла и его расположения.
- **Выход** — завершение работы программы. При наличии несохранённых изменений будет предложено их сохранить.

## 2. Пункт меню «Правка»

- **Отменить** — отмена последнего действия в области редактирования текста.
- **Повторить** — повтор последнего отменённого действия.
- **Вырезать** — вырезание выделенного фрагмента текста в буфер обмена.
- **Копировать** — копирование выделенного фрагмента текста в буфер обмена.
- **Вставить** — вставка текста из буфера обмена.
- **Удалить** — удаление выделенного фрагмента текста.
- **Выделить всё** — выделение всего текста в области редактирования.

## 3. Пункт меню «Справка»

- **Вызов справки** — отображение данного окна с описанием работы программы.
- **О программе** — информация о версии программы и разработчике.
